# Development Roadmap — KaziX

## Phase 1: Foundation (Current)
- [x] **Project Structure:** Establish root directories.
- [x] **Documentation:** PRD, tasks, AI agent instructions.
- [ ] **Backend Scaffolding:** Initialize Node.js/Express, setup routing.

## Phase 2: Core Functionality (Upcoming)
- [ ] **Authentication:** Register/login for clients and fundis.
- [ ] **Job Flow:** Posting jobs, applying for jobs, and hiring fundis.
- [ ] **Dashboard Integration:** Connect frontend dashboards to backend APIs.
- [ ] **Profile Management:** Edit worker profiles and client details.

## Phase 3: Trust & Payments
- [ ] **ID Verification:** Implement KYC processes for fundis.
- [ ] **M-Pesa Escrow:** Integrate Daraja API for STK Push and escrow logic.
- [ ] **Escrow Release:** Logic for releasing funds upon job completion.
- [ ] **Review System:** Post-job feedback and rating updates.

## Phase 4: Communication & Notifications
- [ ] **SMS Integration:** SMS alerts for job postings and applications.
- [ ] **In-App Messaging:** Secure messaging between clients and fundis.
- [ ] **System Notifications:** Dashboard alerts for account activity.

## Phase 5: Scaling & Optimization
- [ ] **Performance:** Load testing and frontend optimization.
- [ ] **Analytics:** Monitoring key performance indicators (KPIs).
- [ ] **Mobile App:** Start mobile app development (Flutter/React Native).
