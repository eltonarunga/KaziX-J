# KaziX Project Planning & Milestones

This document outlines the project roadmap and milestones for KaziX.

## Phase 1: Foundation ✅
- **Project Structure:** Established root directories and core files.
- **Documentation:** PRD, tasks, and AI instructions.
- **Backend Scaffolding:** FastAPI setup, routing, and static frontend serving.

## Phase 2: Core Functionality 🚧
- **Authentication:** Register/login for clients and fundis using Supabase Phone OTP.
- **Job Flow:** Posting jobs, applying for jobs, and hiring fundis.
- **Dashboard Integration:** Connecting frontend dashboards to backend APIs.
- **Profile Management:** Editing worker profiles and client details.

## Phase 3: Trust & Payments
- **ID Verification:** Implementation of KYC processes for fundis.
- **M-Pesa Escrow:** Integration with Daraja API for STK Push and escrow logic.
- **Escrow Release:** Logic for releasing funds upon job completion confirmation.
- **Review System:** Post-job feedback and rating updates.

## Phase 4: Communication & Notifications
- **SMS Integration:** SMS alerts for job postings and applications via Africa's Talking.
- **In-App Messaging:** Secure messaging between clients and fundis.
- **System Notifications:** Dashboard alerts for account activity.

## Phase 5: Scaling & Optimization
- **Performance:** Load testing and frontend performance optimization.
- **Analytics:** Monitoring key performance indicators (KPIs).
- **Mobile App:** Initial development of native mobile applications (Flutter/React Native).
